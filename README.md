

# 📦 Home Inventory Management System (Figma to React)

A frontend job assignment focused on converting a static Figma design into a clean, responsive web interface using **React** and **Tailwind CSS**. 

The main goal of this project was pixel-perfect translation, clean code structure, and accurate component layout according to the provided UI design.

---

## 🎯 Project Goals & Scope

* **Figma-to-Component Fidelity:** Directly translated the layouts, typography, and spacing from the Figma file into reusable React components.
* **Static Interface Focus:** Built entirely as a frontend presentational application using client-side mock data. No backend, database, or complex animations were required for this scope.
* **Component-Driven Layout:** Broken down into interactive tabs, side navigation panels, data tables, and dynamic detail layouts.

## 🛠️ Built With

* **React 19** – Component structure and frontend view layer.
* **Vite** – Fast local development server and build tool.
* **Tailwind CSS v4** – Utility-first CSS framework used for layout and styling.
* **React Router** – Used for basic navigation between the list and detail pages.

## 🚀 How to Run the Project Locally

1. **Clone the project:**
   ```
   git clone [https://github.com/yourusername/home-inventory.git](https://github.com/yourusername/home-inventory.git)
   cd home-inventory

```

2. **Install the node packages:**
```
npm install

```


3. **Start the local dev server:**
```
npm run dev

```
